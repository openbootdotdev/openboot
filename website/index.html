FAQ

Common questions about OpenBoot

OpenBoot supports macOS 12.0 (Monterey) and later, on both Apple Silicon (M1/M2/M3) and Intel Macs. It automatically detects your architecture and installs the correct packages.

OpenBoot is 100% open source. You can inspect the code on GitHub before running. We also support --dry-run to preview exactly what will be installed without making changes.

Yes! Choose from 8 presets tailored for different workflows. Use --preset minimal for a lightweight setup, or --preset full for everything. Run with --dry-run to preview first.

OpenBoot backs up existing dotfiles before making changes. Packages can be removed individually with brew uninstall package-name or in bulk with brew list | xargs brew uninstall.

No. OpenBoot has zero analytics, zero telemetry, and zero network calls except for downloading packages from official sources. Everything runs locally on your machine.
